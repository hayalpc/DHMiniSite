# DHMiniSite
MongoDB configuration in appsettings.json
```
  "DHMiniSiteDatabase": {
    "ConnectionString": "mongodb://localhost:27017",
    "DatabaseName": "dh_mini_site",
    "PostsCollectionName": "posts",
    "CommentsCollectionName": "comments"
  }
```
